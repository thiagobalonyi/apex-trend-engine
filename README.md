# Apex Trend Engine

Algorithmic trading system focused on trend continuation and market structure.

## Strategy Overview

Apex Trend Engine identifies high-probability directional moves using:

- Trend alignment (EMA 200)
- Volatility filtering (ATR)
- Trend strength confirmation (ADX)
- Structured entries with controlled risk

The system is designed to operate consistently across different market conditions, prioritizing risk control and stability over aggressive returns.

## Key Features

- Adaptive trailing stop based on volatility
- Partial profit-taking
- Break-even protection
- Multi-timeframe robustness
